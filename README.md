- 上传文件

- 多选、单选、抽屉、开关等按钮
- ···

## 文件说明

| 文件名 | 文件说明 |
|---|--- |
| profile.js | 方便管理项目相关信息与项目中可能会使用的部分数据 |
| uploads | 后台生成的文件储存位置 |
| BACKUP | 数据库备份 |
| app | 后台程序位置 |
| message.js | 可复用的消息使用中心 |
| err.js | 可复用的页面错误信息提示 |
